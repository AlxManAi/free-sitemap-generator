# Создание иконки приложения

## Автоматическое создание

Для создания иконки приложения из SVG в формат ICO выполните:

```bash
python create_icon.py
```

Это создаст файл `assets/icon/app.ico` с размерами 16x16, 32x32, 48x48, 64x64 и 256x256.

## Требования

Для работы скрипта необходимы:
- `Pillow` (PIL)
- `cairosvg`

Они будут автоматически установлены при первом запуске, если отсутствуют.

## Ручное создание

Если автоматическое создание не работает, вы можете:

1. Открыть `assets/icon/app_icon.svg` в графическом редакторе
2. Экспортировать в PNG с размерами 256x256
3. Использовать онлайн-конвертер или инструмент типа ImageMagick для создания ICO:
   ```bash
   convert app_icon_256.png -define icon:auto-resize=256,128,64,48,32,16 app.ico
   ```

## Проверка иконки

Иконка будет автоматически использована при:
- Запуске приложения (окно приложения)
- Сборке exe через `python setup.py` (иконка exe файла)
